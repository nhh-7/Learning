##### 背包问题

 0/1 背包模板 (物品有限)

```go
// weights: 物品重量, values: 物品价值, W: 背包总容量
func knapsack01(weights []int, values []int, W int) int {
    dp := make([]int, W+1) // 默认初始化为0

    // 遍历物品
    for i, w := range weights {
        v := values[i]
        // 遍历容量：【关键】必须倒序！保证物品只被放入一次，dp[j-w]时上一轮遍历留下来的旧值，这样才确保物品只使用一次
        for j := W; j >= w; j-- {
            dp[j] = max(dp[j], dp[j-w] + v)
        }
    }
    return dp[W]
}
```

完全背包模板（物品无限）

```go
func knapsackUnbounded(weights []int, values []int, W int) int {
    dp := make([]int, W+1)

    // 遍历物品
    for i, w := range weights {
        v := values[i]
        // 遍历容量：【关键】必须正序！因为当前物品可以被多次使用,因为这里用的dp[j-w]这一轮已经更新过了
        for j := w; j <= W; j++ {
            dp[j] = max(dp[j], dp[j-w] + v)
        }
    }
    return dp[W]
}
```

如果题目是求**排列数**还是**组合数**，循环顺序会有变化（常见于完全背包）：

1. **求组合数**（顺序无关，如 `[1, 2]` 和 `[2, 1]` 视为同一种）：
   - **先遍历物品，再遍历背包**，固定物品顺序，避免重复计算排列。
   - 典型题目：LeetCode 518 零钱兑换 II。
2. **求排列数**（顺序有关，如 `[1, 2]` 和 `[2, 1]` 是不同的）：
   - **先遍历背包，再遍历物品**，每一位容量都尝试所有物品，考虑顺序。
   - 典型题目：LeetCode 377 组合总和 Ⅳ。
